# WorkTrackr

A full-stack task management application built with Next.js (Frontend)
and Node.js + Express + Prisma (Backend).

## Features

-   Authentication (JWT)
-   Create, Edit, Delete Tasks
-   Toggle Completion
-   Optimistic UI
-   Keyboard Support
-   Protected Routes

## Tech Stack

Frontend: Next.js, React, Tailwind\
Backend: Node.js, Express, Prisma, PostgreSQL

## Setup

### Backend

cd server\
npm install\
npx prisma migrate dev\
npm run dev

### Frontend

cd client\
npm install\
npm run dev
